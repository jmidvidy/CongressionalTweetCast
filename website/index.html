What is Congressional TweetCast?

Congressional TweetCast is a Computer Science project made at Northwestern Universtiy in Evanston IL.

This is a tool for political and electoral research. How it works is simple: users enter in a twitter handle and select a specfiic congressional election.

This website will then deploy sophisiticated algorithms to analyze the inputted twitter user's tweets, and tell you - the user - which candidate the inputted twitter handle will vote for in the specified election.

Our algorithms are built off tweetpy, machine learning in python (more specifically text analysis through a Naive Bayes Classifier), as well as your standard web tools like HTML, JavaScript, Node.js, and some lovely animations.

Please don't abuse this tool.
